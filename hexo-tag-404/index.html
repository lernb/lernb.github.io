Hexo标签或分类出现404

ler
2021-01-10  

今天部署博客发现部分标签点击出现404问题，许久之前也出现过类似现象，把字母改成小写就正常了。

这次的问题是之前正常的标签也出现了404，改成小写果然又正常了。看来问题应该就是出在字母的大小写上了。

于是乎谷歌一阵搜索，果然，导致这个问题的原因就是Hexo 中的Git 配置文件设置了对大小写不敏感。

解决方法

在Hexo 博客目录下的.deploy_git目录中，再进入.git目录（此目录默认是隐藏的，需要系统设置打开显示隐藏文件夹），修改此目录中的config文件:

# 将以下条目设置成false即可
ignorecase = false

保存，然后清除并重新生成部署博客即可。
