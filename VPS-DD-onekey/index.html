VPS DD安装Linux/Windows系统

ler
2020-07-16   源挖站否

安装脚本不建议用于生产环境

Linux VPS主机一键全自动DD安装纯净Debian/Ubuntu/CentOS和Windows操作系统，Linux的ISO建议直接到官网上下载，而Windows由于版权的问题一般来说可以使用热心网友制作的DD包。

安装Linux系统

全自动安装后root密码：MoeClub.org，安装完成后最好修改为自己的密码。

全自动安装CentOS时默认提供VNC功能，可使用VNC Viewer查看进度，VNC端口为1或者5901，系统安装后自动删除。

目前CentOS系统只支持任意版本重装为CentOS 6.x及以下版本。注意：OpenVZ构架不适用。

安装脚本

#Debian/Ubuntu系统:
apt-get update
 
#RedHat/CentOS系统:
yum update
 
#安装必需软件:
#Debian/Ubuntu:
apt-get install -y xz-utils openssl gawk file
 
#RedHat/CentOS:
yum install -y xz openssl gawk file
 
#下载InstallNET.sh安装脚本：
wget --no-check-certificate -qO InstallNET.sh 'https://moeclub.org/attachment/LinuxShell/InstallNET.sh' && chmod a+x InstallNET.sh
 
#脚本使用方法：
bash InstallNET.sh   -d/--debian [dist-name]
                     -u/--ubuntu [dist-name]
                     -c/--centos [dist-version]
                     -v/--ver [32/i386|64/amd64]
                     --ip-addr/--ip-gate/--ip-mask
                     -apt/-yum/--mirror
                     -dd/--image
                     -a/-m
 
# dist-name: 发行版本代号
# dist-version: 发行版本号
# -apt/-yum/--mirror : 使用定义镜像
# -a/-m : 询问是否能进入VNC自行操作. -a 为不提示(一般用于全自动安装), -m 为提示。

使用脚本自动安装Debian

#下载InstallNET.sh安装脚本（如已下载则跳过）：
wget --no-check-certificate -qO InstallNET.sh 'https://moeclub.org/attachment/LinuxShell/InstallNET.sh' && chmod a+x InstallNET.sh
 
#全自动安装debian 8 64位
bash InstallNET.sh -d 8 -v 64 -a --mirror 'http://mirrors.ustc.edu.cn/debian/'
 
#全自动安装debian 9 64位
bash InstallNET.sh -d 9 -v 64 -a --mirror 'http://mirrors.ustc.edu.cn/debian/'

使用脚本自动安装CentOS

#下载InstallNET.sh安装脚本（如已下载则跳过）：
wget --no-check-certificate -qO InstallNET.sh 'https://moeclub.org/attachment/LinuxShell/InstallNET.sh' && chmod a+x InstallNET.sh
 
#全自动安装CentOS 6.10 64位
bash InstallNET.sh -c 6.10 -v 64 -a --mirror 'http://mirror.centos.org/centos'
 
#全自动安装CentOS 7 64位（目前该脚本不支持）
bash InstallNET.sh -c 7 -v 64 -a --mirror 'http://mirror.centos.org/centos'

使用脚本自动安装Ubuntu

#下载InstallNET.sh安装脚本（如已下载则跳过）：
wget --no-check-certificate -qO InstallNET.sh 'https://moeclub.org/attachment/LinuxShell/InstallNET.sh' && chmod a+x InstallNET.sh
 
#全自动安装ubuntu 16.10 64位
bash InstallNET.sh -d 16.10 -v 64 -a --mirror 'http://archive.ubuntu.com/ubuntu/'
 
#全自动安装ubuntu 18.10 64位
bash InstallNET.sh -u 18.10 -v 64 -a --mirror 'http://archive.ubuntu.com/ubuntu/'

经过使用测试，本文介绍的安装脚本最好在IO读写较快的机器上使用，否则可能会出现卡死的情况。

已在腾讯云轻量应用服务器通过实际测试。

纯净的Debian/Ubuntu/CentOS与Windows系统安装大概需要花费半个小时的时间（实际视硬件与网络环境因素而定）
